## 프로젝트 개요

Tactile Games게임사의 캐쥬얼게임인 **Cookie Cats**을 이용해 A/B 테스트 데이터를 진행했습니다.

초기 **게이트(Level 30 vs Level 40)** 위치가 **유저 이탈과 유지율(D1,D7)** 에 어떤 영향을 미치는지 분석하는 프로젝트입니다.

데이터는 Kaggle에서 제공되는 공개 데이터셋을 사용했습니다.
https://www.kaggle.com/datasets/farshadbeheshtinia/cookiecats
## 데이터셋 정보

  컬럼             설명
  ---------------- -------------------------------------
  userid           사용자 고유 ID
  version          A/B 테스트 버전 (gate_30 / gate_40)
  sum_gamerounds   플레이한 총 게임 라운드 수
  retention_1      1일 후 유지 여부 (0/1)
  retention_7      7일 후 유지 여부 (0/1)

## 분석 목표

-   게이트 위치 변경이 유지율에 미치는 영향 파악
-   D1, D7 유지율 비교
-   초기 난이도 변화가 이탈에 주는 영향 분석
-   비즈니스 관점 인사이트 도출

## 분석 과정 요약

1.  데이터 로드 및 구조 확인
2.  EDA
3.  이상치 제거 및 전처리
4.  A/B 테스트 통계 분석
5.  결론 및 시사점 도출

## 주요 결과 요약

-   gate_30 유저는 평균 게임 플레이 수가 더 많음
-   gate_40은 초기 이탈 수 증가
-   D1 유지율: gate_30 > gate_40
-   D7 유지율 차이는 미미하거나 데이터에 따라 다름

## 결론 & 인사이트

-   gate_40은 초기 난이도 차이로 이탈 유발
-   retention 지표 개선을 위해 gate_30 구성 추천
-   향후 개선 방향: 난이도 조정, 보상 강화, 튜토리얼 개선
